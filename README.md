### 💫 프로젝트 소개

---

- 영화를 의미하는 "cinema🎞️"와 “빠른⚡”을 의미하는"wiki"를 합성하여 만든 "Cinewiki"는 영화에 대한 정보를 다양한 메뉴를 통해 확인 할 수 있는 서비스입니다.

### 📝 지원 기능

- 영화 정보 조회 (공용 모달처리)
- 영화 리스트 보기 (무한 스크롤)
- 영화 검색 기능

### 📅프로젝트 기한 및 내용

2024.2.28 ~ 2024.3.13 (+이후 리팩토링 진행)
